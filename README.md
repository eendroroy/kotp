Kotlin(/Java) OTP generation library.

### Installation

*Maven*

```xml
<dependency>
	<groupId>com.github.eendroroy</groupId>
	<artifactId>kotp</artifactId>
	<version>0.1.2</version>
</dependency>
```

*Gradle*
```groovy
dependencies {
    implementation 'com.github.eendroroy:kotp:0.1.2'
}
```
